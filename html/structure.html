webpages can be divided into 4 parts:

  • header that is similar on every page, and is the main navigation for the entire site
  • main content that changes for every page, a list of articles (basically an about page)
  • sidebar that links to archives and categories
  • footer for additional links to less important pages
There are some elements you can use as containers for other elements

Header
acts as the intro to the webpage, with the logo, tagline and navigation links

Footer
as opposed to the header, the footer is usually the last element of a page,
where the main navigation links are repeated and secondary ones are added.

Main
The main element contains the most important content of the page that define the purpose of the page.
while all webpages contain common elements (like the header, and navigation), the main element focuses on unique content.

Aside
The aside element usually lives alongside the main and contains additional info related to the main content.

Section
The section element allows to group.
