CSS = Cascading Style Sheet
its used to style markup languages because plain text looks bad
therefore css is worthless on its own.
It brings an html doc to life, you can choose fonts, apply colours, define margins, position elements,
animate interactions and a lot more.

How CSS works
you select the html element (like a paragraph), choose a property to alter (like colour),
and apply a certain value to the property (like red):
(p{ color: red;})

Where to write CSS:
CSS is an attribut
you can write css directly on an html element by using the (style) attribute:
(<p style=color: red;>) --> this will give red text

CSS in the <head>
you can use <style> in the <head>

CSS in a separate file:
you can write css in a separate file with a .css extension and then link it to your
html by using the <link> html tag
people usually use this method (in a different file)

The reason why ppl dont css directly in html is because it looks like ass and ppl have decency (usually)
it also makes maintenance easier, the same css file can be used for a whole website
it provides flexibility: focus on the content on one side (html), and style on the other (css).
